# How to run RPN Calculator

1. Unzip To_HW2.zip.
2. Open a terminal window and cd into the folder named assignment2.
3. mkdir bin
4. javac -sourcepath src src/Assignment2.java -d bin
5. cd bin
6. java Assignment2
7. You will be prompted to enter a postfix expression.
8. Enter a postfix expression  and then press "enter".
9. You will either get an error message or the result of the postfix calculation.
10. You will be asked if you want to continue or not. You can type 'y' and press "enter" to continue.
  - Note that 'y' is case-sensitive, so you must use lowercase.
  - Pressing any other key will end the program.
