# How to run RPN Calculator

1. Run the program.
2. Enter a postfix expression when prompted and then press "enter".
3. You will either get an error message or the result of the postfix calculation.
4. You will be asked if you want to continue or not. You can type 'y' and press "enter" to continue.
  - Note that 'y' is case-sensitive, so you must use lowercase.
  - Pressing any other key will end the program.
